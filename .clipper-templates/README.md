# Obsidian Web Clipper Templates

Templates for the [Obsidian Web Clipper](https://obsidian.md/clipper) browser extension.

## Templates

23 clipper templates in `.clipper-templates/templates/` for sites like YouTube, GitHub, Wikipedia, Goodreads, Letterboxd, IMDB, Spotify, NYT Cooking, Google Maps, and more.

## Testing

```bash
npm install
npm test
```

Tests extract ALL properties from each template JSON and verify they can be extracted from real pages.

## Usage

Import templates into Obsidian Web Clipper:

1. Open extension settings
2. Import from `.clipper-templates/templates/*.json`
